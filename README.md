# lucamannella.github.io

Personal website of Luca Mannella, hosted on GitHub Pages.
To see the website go to: https://lucamannella.github.io

## 📁 Project Structure

```
├── index.html       # Main page (semantic HTML)
├── style.css        # Stylesheet (clean, minimal design)
└── README.md        # This file
```

## 🏗️ Current Structure

### `index.html`
- **Header** with sticky top navigation
- **Navigation links**: Home, Publications, CV, Contact
- **Hero section** with introduction and tagline
- **Content sections** with placeholder "Website in Progress"
- **Footer** with copyright info
- Semantic HTML5 structure (`header`, `nav`, `main`, `section`, `footer`)

### `style.css`
- Clean academic/professional aesthetic
- Neutral color palette
- Responsive design (adapts to 768px and 480px breakpoints)
- System fonts for performance
- Simple hover effects on navigation
- Centered content with 900px max-width

## 🚀 Getting Started

This is a static website using plain HTML and CSS with **no frameworks** (no React, Bootstrap, Tailwind, Jekyll, etc.).

1. **View locally**: Open `index.html` in a browser
2. **Deploy**: Push to GitHub; site will be live at https://lucamannella.github.io
